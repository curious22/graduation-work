# Project for graduate work

## Requirements
- Python 3
- Scrapy 1.2.1
- MongoDB 3.4.0
- Redis
- Falcon 1.1.0

## Run

Navigate into `falcon_api/`

Run command `gunicorn app`

```
sudo apt-get install build-essential libssl-dev libffi-dev python-dev python3-dev
```

> API's endpoints are [here](docs/endpoints.md)
